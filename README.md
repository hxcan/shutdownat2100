# 21 点关机应用

Android 自动关机工具，帮助用户在每晚 21:00 自动关机，养成良好的作息习惯。

## 🎯 功能特性

- ⏰ **定时关机**：每晚 21:00 自动关机
- 🔒 **备用方案**：若无法获取关机权限，则锁住手机并建议用户休息
- 💤 **健康提醒**：温馨提示用户按时休息
- 📱 **轻量简洁**：无广告、无多余权限

## 🚀 使用方法

1. 安装应用
2. 授予必要的权限（关机权限或设备管理器权限）
3. 应用会在每天 21:00 自动执行关机或锁屏操作

## ⚠️ 权限说明

- **关机权限**：部分手机需要系统签名或 Root 权限
- **设备管理器**：用于锁屏备用方案
- **通知权限**：用于显示关机提醒

## 📋 注意事项

- 不同品牌手机的关机权限可能不同
- 如无法获取关机权限，应用会自动切换到锁屏模式
- 建议配合充电使用，避免电量耗尽

## 🔧 技术栈

- Android SDK
- Java/Kotlin
- Device Admin API
